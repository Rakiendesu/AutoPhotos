# AutoPhotos
Typecho的图片瀑布流效果插件
基于<a href="https://github.com/nk-o/flickr-justified-gallery#wordpress-plugin">Flickr's Justified Gallery</a>的瀑布流图片效果<br>
前AutoPhotos2.0.0(泽泽站长）修改版
## 更新
 - 2.1.1  添加了图像延迟加载 (适用typecho-butterfly)
